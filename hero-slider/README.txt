  Návod na zmenu obrázkov / How to change images
═══════════════════════════════════════════════

SLOVENSKY:
──────────
Ak chcete zmeniť obrázky v úvodnom slideri na stránke:

1. Nahrajte svoje obrázky do tohto priečinka cez FTP
2. Pomenujte ich: slide-1.jpg, slide-2.jpg, slide-3.jpg, ...
3. Formát: JPG (odporúčaná veľkosť: 1920x1080px alebo väčšie)
4. Maximálny počet: 10 obrázkov
5. Stránka ich načíta automaticky — nič iné nemusíte meniť!

Príklad:
  hero-slider/
    slide-1.jpg  ← prvý obrázok
    slide-2.jpg  ← druhý obrázok
    slide-3.jpg  ← tretí obrázok (voliteľný)

Tip: Pre najlepší výsledok používajte horizontálne (landscape) 
fotografie s rozlíšením aspoň 1920px na šírku.


ENGLISH:
────────
To change the hero slider images on the homepage:

1. Upload your images to this folder via FTP
2. Name them: slide-1.jpg, slide-2.jpg, slide-3.jpg, ...
3. Format: JPG (recommended size: 1920x1080px or larger)
4. Maximum: 10 images
5. The website loads them automatically — no code changes needed!

Example:
  hero-slider/
    slide-1.jpg  ← first image
    slide-2.jpg  ← second image
    slide-3.jpg  ← third image (optional)

Tip: For best results, use horizontal (landscape) photos 
with at least 1920px width.
